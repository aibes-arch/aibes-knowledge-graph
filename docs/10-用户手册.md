# 10 用户手册

## 10.1 准备工作

1. 确保后端服务已启动：`http://localhost:8000`
2. 确保 Neo4j 已启动并可连接
3. 打开前端页面：`frontend/index.html` 或 `http://localhost:8000/static/index.html`

## 10.2 上传文档

1. 在左侧面板点击「选择文件」
2. 选择 PDF、Word、Markdown 或 TXT 文件
3. 点击「1. 上传文档」
4. 状态栏显示上传成功及文档 ID

## 10.3 解析文本

1. 上传成功后，「2. 解析文本」按钮变为可用
2. 点击「2. 解析文本」
3. 系统会自动提取文本并按句子合并切块
4. 状态栏显示文本长度和切块数量

## 10.4 抽取知识

1. 解析完成后，「3. 抽取知识」按钮变为可用
2. 点击「3. 抽取知识」
3. 系统调用 LLM（或 Mock 抽取器）从每个文本块中抽取实体和关系
4. 状态栏显示候选数量

> 如果未配置 LLM Key，可在后端 `.env` 中设置 `MOCK_LLM=true` 使用演示模式。

## 10.5 写入图谱

1. 抽取完成后，「4. 写入图谱」按钮变为可用
2. 点击「4. 写入图谱」
3. 系统将候选实体和关系 MERGE 到 Neo4j
4. 写入完成后可查看统计信息

## 10.6 查看知识图谱

### 加载全图

1. 点击「加载全图」
2. 中间区域显示所有实体和关系
3. 可拖拽节点、滚轮缩放、点击节点查看详情

### 搜索节点

1. 在搜索框输入关键词，如「风机」
2. 点击「搜索节点」
3. 匹配的节点以列表形式显示
4. 点击节点可展开其邻居关系

### 查看节点详情

1. 点击图谱中的任意节点
2. 右侧面板「节点详情」显示该节点的完整属性

## 10.7 知识问答

1. 在右侧「图谱问答」输入框输入问题
2. 例如：「风机异响如何处理？」
3. 点击「提问」
4. 系统基于图谱上下文调用 LLM 生成答案
5. 回答下方显示引用的实体来源

## 10.8 示例文档

项目根目录提供了 `sample_equipment.md`，内容如下：

```markdown
# 风机冷却系统维修手册

## 装备概述

装备名称：主轴冷却风机
子系统：冷却系统
部件：冷却风机、散热风扇、电机轴承

## 常见故障

冷却风机出现异响，可能是电机轴承磨损导致。
风机不转通常由电源模块故障引起。

## 维修方案

针对电机轴承磨损，需要更换电机轴承。

## 工具与备件

本维修方案需要以下工具：内六角扳手、万用表。
需要备件：电机轴承、散热风扇。
```

使用该文档可快速体验完整流程。

## 10.9 注意事项

- 实体对齐功能在 MVP 中未实现，可能出现「风机」和「冷却风机」被识别为不同实体的情况
- 复杂 PDF（扫描件、复杂排版）可能需要 OCR 支持
- LLM 抽取质量取决于模型能力和 Prompt 设计
